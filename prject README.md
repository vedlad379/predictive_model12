 Predictive Analytics Using Historical Data

 Project Overview

This project focuses on predictive analytics using historical sales data. The goal is to analyze past trends and forecast future sales using machine learning and visualization tools.

The project is built using:

* Python
* Machine Learning
* Microsoft Power BI


 Features
 
* Data cleaning and preprocessing
* Historical trend analysis
* Sales forecasting
* Regression-based prediction
* Interactive dashboard in Power BI
* KPI visualization
* Forecast visualization


Technologies Used

* Python
* pandas
* matplotlib
* scikit-learn
* Power BI
* Excel
